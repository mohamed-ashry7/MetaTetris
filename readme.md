# MetaTris
The game is ready to be played into the terminal
the controllers are
```
a -> rotate left

d -> rotate right 

down arrow -> go down

right arrow -> go right

left arrow -> go left

space -> hard drop

p -> for pausing 

q-> for quiting
```
